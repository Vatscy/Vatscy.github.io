## コマンド一覧

### 新規記事作成

```zsh
% rake new_post
```

- /source/_posts内に新たなMarkdownファイルが生成される。

### ブログページ生成

```zsh
% rake generate
```

### プレビュー

```zsh
% rake preview
```

### デプロイ

```zsh
% rake deploy
```

#### ※下記コマンドを使用することで生成とデプロイを同時に行うことも可能

```zsh
% rake gen_deploy
```
